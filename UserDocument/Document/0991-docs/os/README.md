## 框架说明
>Grace是一个轻型组件式PHP框架，尝试用优雅的代码编写，适用于简单系统的快速开发，提供了简单的路由方式，简单且实用。内置所有的组件都可以单独使用，并且附有测试代码和全文档,有很好的安全性，稳定性，扩展性



## 亮点

- 简单的API
- [RFC3986](http://tools.ietf.org/html/rfc3986) 编码规范
- 遵循[PSR-7][]日志接口
- 全文档支持[文档]()
- 组件框架无关[组件可以分离出来单独使用,有独立的文档和测试代码支持]；
- Composer支持
- 遵循[PSR-2][]和[PSR-4][]规范
- 单元测试
- 轻量级框架 

## 系统需求

- **PHP >= 5.5.0** , 请使用稳定版本
- `mbstring` 扩展
- `pdo` 扩展
- `curl` 扩展
- "league/url": "^3.3"
